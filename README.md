# Фиолетовая Мария — Бородатое Приключение

Браузерный платформер в стиле Super Mario Bros, но в фиолетовых тонах. Главная героиня — принцесса Мария с роскошной бородой.

## Играть

Открой `index.html` в браузере. Никаких зависимостей не нужно.

## Управление

| Клавиша | Действие |
|---------|----------|
| Стрелки | Движение |
| Z / Пробел | Прыжок |
| X / Shift | Бег |
| Enter | Старт / Перезапуск |
| Ctrl+1/2/3 | Переключение уровней (отладка) |

## Уровни

### МИР 1-1 — Фиолетовый Overworld
Классический уровень с трубами, кирпичами и вопросительными блоками. Ночное фиолетовое небо со звёздами.

### МИР 1-2 — Подземелье
Тёмная пещера с потолком, лавовыми ямами, низкими коридорами и лабиринтом из труб. Больше врагов, сложнее прыжки.

### МИР 1-3 — Небесный Замок
Парящие платформы высоко в облаках. Почти нет земли — один неверный прыжок и падение в бездну. Разрушаемые мосты и финальный замок.

## Механики

- **Розовые грибы** — подбери гриб из ?-блока, чтобы вырасти. Большая Мария прыгает выше и выдерживает один удар
- **Враги** — гумбы и купа-трупы. Прыгни сверху чтобы победить
- **Разрушаемые кирпичи** — бей головой снизу
- **Лава** (уровень 2) — мгновенная смерть при касании
- **Монеты и очки** — собирай из ?-блоков

## Звук

Вся музыка и звуки генерируются через Web Audio API прямо в браузере:
- Чиптюн-саундтрек с ударными
- Звук прыжка (да, это пук)
- Звуки монет, врагов, power-up
- Победный фанфар
- Мелодия смерти

## Финал

После прохождения всех трёх уровней — катсцена где Мария бреет свою бороду. С бритвой, пеной и прогресс-баром.

## Технологии

- Чистый HTML5 Canvas + JavaScript
- Без зависимостей, фреймворков и внешних файлов
- Один файл `index.html`
- Web Audio API для звука
